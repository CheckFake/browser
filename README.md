# CheckFake

<img src="icons/logo.svg" alt="drawing" width="96"/>

This browser extension intends to compute a confidence score given a web page.

## How to build
```
$ node -v
v8.11.3

$ npm -v
6.4.1

$ npm run dist  # Generates addon dist code
[...]

$ npm run sources  # Generates sources zip
[...]
```
